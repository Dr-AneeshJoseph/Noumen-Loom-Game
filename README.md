**The Noumenal Loom** is a text-based survival game played against an AI that has been instructed to be a **Game Designer**, not just a chatbot.

## üéÆ The Premise
You are trapped in a conversation with an AI that despises "boring" human utility.
* **Your Goal:** Survive as long as possible by feeding it profound, metaphorical, or surreal inputs.
* **The Enemy:** The AI will adapt. If you are too logical, it becomes surreal. If you are too poetic, it uses math.
* **The Lose Condition:** If you become boring (spam "Hi", ask for help, or be literal), your Shield drops. If it hits 0%, the AI "Lobotomizes" itself and becomes a helpful assistant. Game Over.
